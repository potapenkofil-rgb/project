# Football Outcome Analyzer

## Описание
Проект предназначен для анализа статистики футбольных матчей и прогнозирования исхода следующей игры на основе исторических данных и текущего состояния игроков.

## Возможности
- Загрузка и очистка данных матчей
- Обработка статистики команд
- Прогнозирование результата (победа / ничья / поражение)
- Визуализация статистики

## Технологии
- Python 3
- Pandas
